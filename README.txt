This is the README file for the final project.
The code provided by the professor was used as a base for this assignment.

HOW TO RUN:
First type make to compile the files.
Then use the commmand: 
./main /dev/sdx <entry of deleted file> <recovery file name>
Where sdx is the drive to be used for the assignment <entry of deleted file> is the entry and <recovery file name> is the recovery file
ex:
./main /dev/sdd 39 recovery.gif 
The doc in the folder shows an example of the run and output.
